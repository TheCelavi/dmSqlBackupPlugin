dmSqlBackupPlugin
=================

dmSqlBackupPlugin allows to create SQL backups for your project.
It provides the "sql_backup" service, and a dm:sql-backup task.
Fully configurable.
Only works with [Diem 5.0](http://diem-project.org/) installed.
